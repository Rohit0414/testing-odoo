Your Questions, Answered

How easy is the setup?

It's a standard Odoo module install. Plug in your BQ Project ID, Dataset ID, and Service Account JSON in Settings. You're live in minutes.

Which Odoo versions are supported?

The connector supports Odoo 15, 16, 17, and 18. Our Pro Plus plan includes free upgrades for all future Odoo versions.

How does it handle millions of records?

It intelligently exports data in configurable batches to prevent timeouts and server strain. We've successfully tested it on tables with over 8 million records.

When should I choose the Pro Plus version?

Choose Pro Plus if you need automated, incremental updates (syncing only changes), scheduled refreshes, support for future Odoo versions, or are working with very large, active datasets.

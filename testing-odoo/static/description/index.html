Frequently Asked Questions

What do I need to use this module?

An Odoo instance, a Google Cloud account with BigQuery enabled, and a Service Account with "BigQuery Data Editor" and "Job User" roles.

How is the sync scheduled?

You can trigger exports manually or automate them using Odoo's built-in "Automated Actions" (cron jobs), allowing you to schedule syncs daily, hourly, or at any interval you choose.

Can it sync data from custom Odoo modules?

Yes. The connector can access and export data from any standard or custom model within your Odoo database, including any custom fields you've added.

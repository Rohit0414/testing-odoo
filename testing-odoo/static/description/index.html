Frequently Asked Questions

How easy is the setup?

Standard Odoo module install. Enter your BQ Project ID, Dataset ID, and Service Account JSON in Settings. You're done in minutes.

Which Odoo versions are supported?

The connector supports Odoo 15, 16, 17, and 18. The Pro Plus plan includes free upgrades for future versions.

How does it handle millions of records?

It intelligently exports in configurable batches to avoid timeouts and server strain. Tested on tables with over 8 million records.

When do I need the Pro Plus version?

Choose Pro Plus for automated, incremental updates (syncing only changes), scheduled refreshes, free Odoo upgrades, and support for very large datasets.

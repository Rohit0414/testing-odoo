SMART DATA MAPPING

Intelligent Schema Handling

Our connector automatically translates complex Odoo fields into clean, query-ready columns in BigQuery, saving you hours of data wrangling.

  • Many2one Fields: Automatically exports the related record's name and ID (e.g., `partner_id`, `partner_name`).
  • Selection Fields: Converts internal keys to human-readable labels (e.g., `state: 'draft'` becomes `state: 'Draft'`).
  • Date & Datetime: Ensures proper ISO formatting for seamless use in BigQuery time-series analysis.

SETUP IN MINUTES

Installation & Configuration

From install to your first data sync in just a few clicks.

  1. Get the module from the Odoo App Store.
  2. Install by adding to your custom addons directory or through Odoo.sh.
  3. Create a Google Cloud Service Account with "BigQuery Data Editor" and "BigQuery Job User" roles.
  4. Upload your Service Account JSON key in Settings → BigQuery Connector.
  5. Configure your first export model, select your columns, set a filter, and run!

FAQ

Frequently Asked Questions

What do I need to use this module?

You will need an active Odoo instance (v14 or later), a Google Cloud account with BigQuery enabled, and a Service Account from that project with "BigQuery Data Editor" and "BigQuery Job User" permissions.

How is my data kept secure?

Data security is paramount. The connector establishes a direct, encrypted connection between your Odoo server and Google's BigQuery API. Your data is never routed through or stored on any third-party servers. Access is controlled entirely by the Google Cloud Service Account key that you provide and manage.

Does it work with my custom Odoo modules and fields?

Yes. The connector dynamically reads your Odoo models, including any custom modules you have installed and custom fields you have added. You can select these custom fields for export just like standard Odoo fields.

What are the costs involved?

You purchase the Odoo to BigQuery Connector module license one-time from the Odoo App Store. The only ongoing operational costs are for your usage of Google BigQuery (for storage and querying), which is billed directly by Google Cloud. There are no additional recurring subscription fees from us.

Can I schedule automatic exports?

Yes. The module leverages Odoo's built-in automation server (Scheduled Actions). You can configure any export profile to run on a schedule you define, such as hourly, daily, or weekly, enabling fully automated, incremental data syncs.

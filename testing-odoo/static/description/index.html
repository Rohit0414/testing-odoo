Features

Export Any Odoo Table

Select any standard Odoo table (like Sales Orders, Invoices, Products, Contacts) and push its data to BigQuery effortlessly. Works with all native and custom modules, giving you complete access to your business data for analytics.

Precise Record Filtering

Use standard Odoo domain filters [('field', 'operator', 'value')] to select exactly which records to export. Only send the data you actually need, reducing transfer volumes and keeping your BigQuery instance lean and efficient.

Specific Column Selection

Choose exactly which fields to send to BigQuery instead of exporting entire tables. This targeted approach accelerates exports and directly reduces your BigQuery storage and query costs.

Handles Large Tables

Our intelligent batching system exports data in manageable chunks, reliably handling tables with 1M+ records. Prevents timeouts and keeps your Odoo system stable even during massive exports.

Incremental (CDC) Exports (Pro Plus Only)

Incremental Change Data Capture (CDC) exports ensure that only new or updated records are sent to BigQuery after the initial full export. This minimizes data transfer, reduces costs, and keeps your BigQuery dataset up-to-date with the latest changes in Odoo. Ideal for businesses requiring frequent updates without re-exporting entire tables.

💡 Tips, Tricks & Positive Insights

  • 💡
    Filter Aggressively: The more specific your Domain Filter, the faster the export and the lower your BQ costs. Don't pull data you don't need today.
  • 📊
    Select Columns Wisely: Every extra column adds time and cost. Only select columns required for your specific BigQuery analysis or reports.
  • 💸
    Understand BigQuery Pricing: Remember BigQuery charges for storage and compute (queries). This connector helps minimize both by sending less data.

Frequently Asked Questions

↠ How easy is setup? ♥
Standard Odoo module install. Enter your BQ Project ID, Dataset ID, and Service Account JSON in Settings. Done.
↠ What support is included? ♥
21 days of free email support for installation and configuration issues for the Pro version.
↠ Which Odoo versions are supported? ♥
Check the compatible versions listed on this page (currently supports Odoo 15.0, 16.0, 17.0, 18.0). Newer Odoo versions require a separate purchase for the Pro license.
↠ How does it handle millions of records? ♥
It exports in batches (configurable size) to avoid timeouts and memory issues. For very large data, consider the Pro Plus variant.
↠ Does Pro handle new records? ♥
Yes. The Pro version exports based on your filters each time. The full filtered table is exported on each run. True incremental export (CDC) is a Pro Plus feature after the initial full export, only new or updated records are synced.
↠ When do I need Pro Plus? ♥
If you work with very large, frequently changing datasets and need only changes synced automatically, or require scheduled/near real-time updates, or want free upgrades for future Odoo versions.

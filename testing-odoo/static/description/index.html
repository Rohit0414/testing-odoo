Frequently Asked Questions

How easy is the setup?

Standard Odoo module install. Enter your BQ Project ID, Dataset ID, and Service Account JSON in Settings. Done.

What support is included?

21 days of free email support for installation and configuration issues for the Pro version. Pro Plus includes 3 months.

Which Odoo versions are supported?

Check the compatible versions on the store page (currently supports 15, 16, 17, 18). Pro Plus includes free upgrades for future versions.

How does it handle millions of records?

It exports in configurable batches to avoid timeouts. For very large data (multi-million records) or frequent syncs, the Pro Plus variant is highly recommended.

When do I need Pro Plus?

If you need automated, incremental updates (syncing only new/changed data), scheduled refreshes, support for future Odoo versions, or are working with very large datasets.
